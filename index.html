Sam-to-bam

Converts the format of a genome file from SAM to BAM

HOW TO BUILD

Step 1. Download a script

$ wget https://github.com/t-ogasawara/sam-to-bam/raw/master/build.sh

Step 2. Run the script on the directory where you put the source files and compile them

$ bash build.sh

HOW TO RUN

SAM-to-BAM

$ build/samtools/samtools sam2bam -@184 -Obam -oout.bam in.sam

Command-line options:

184           (physical_cores - 1) * threads_per_core (e.g, 184 = (24 - 1) * 8)
out.bam       output BAM file name
in.sam　　      input SAM file name
